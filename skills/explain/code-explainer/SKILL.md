# code-explainer

Explains code in plain English by analyzing structure and patterns

## When to use

Use this skill when you need to explains code in plain english by analyzing structure and patterns.

## Key method

`constructor(input)` — runs the code-explainer analysis and returns a structured result.

## Example

```js
const CodeExplainer = require('./index.js');
const skill = new CodeExplainer();
const result = skill.constructor(input);
console.log(result);
```
